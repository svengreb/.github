<p align="center"><a href="https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file" target="_blank">Community health files</a> for <a href="https://github.com/svengreb" target="_blank">@svengreb</a> projects.</p>

This repository provides default [community health files][1] for [@svengreb][2] projects and all its repositories.
This includes all [supported files][3] that can be used universally across repositories:

- **[`issue_template/config.yml`][15]** — The default [configuration file for the issue template chooser][16] in @svengreb repositories.
- **[`code_of_conduct.md`][4]** — The [code of conduct][5] that defines @svengreb project community standards and outlines procedures for handling abuse.
- **[`contributing.md`][6]** — General guidelines to communicate [how people should contribute][7] to projects in order to [build a welcoming community][8].
- **[`funding.yml`][9]** — The default [configuration file for the sponsor button][10] in @svengreb repositories.
- **[`security.md`][11]** — The [security][12] policy with instructions for how to [report security vulnerabilities][13] in @svengreb projects.
- **[`support.md`][14]** — Instructions for how to ask for help and general @svengreb project community & support resources.
